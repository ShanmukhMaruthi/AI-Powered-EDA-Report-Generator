# AI-Powered EDA Report Generator

## Overview

An end-to-end analytics automation platform that transforms raw CSV datasets into professional business intelligence reports.

Users upload a CSV dataset through an n8n form interface. The workflow automatically performs Exploratory Data Analysis (EDA), generates statistical insights using Gemini AI, creates a visually rich HTML report, converts it into PDF format, and delivers the final report directly to the user's email inbox.
